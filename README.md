# Reels Scheduler Bot

This repository contains the full Reels automation setup, divided into two main parts:
1. **[instagram-scheduler-bot](file:///d:/Reels%20automation/instagram-scheduler-bot)**: A NestJS backend that handles scheduling, Supabase interactions, Slack notifications, and Cloudflare R2 uploads.
2. **[frontend](file:///d:/Reels%20automation/frontend)**: A React + TypeScript + Vite web dashboard for managing accounts, posting slots, and viewing the queue.
